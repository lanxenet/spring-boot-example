# 微服务例子
包括服务和UI部分

## 启动

1. 编译

```bat
> cd spring-boot-demo
> mvn clean install
```

2. 启动微服务
```bat
> cd spring-boot-demo
> mvn spring-boot:run
```

3. 启动开发服务
```bat
> cd demo-ui
> yarn install
> yarn run start
```

## 目录结构

```
.
├── /demo-ui/                     # UI
├── /spring-boot-demo/            # 服务

```
